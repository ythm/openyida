# 生日祝福小游戏 需求文档

## 功能需求

### 核心功能

1. **欢迎界面**
   - 输入寿星姓名
   - 输入送祝福人姓名
   - 开始游戏按钮

2. **点蜡烛游戏**
   - 显示生日蛋糕和蜡烛（10根）
   - 点击蜡烛可以吹灭(使用麦克风)
   - 计时功能 - 记录吹灭所有蜡烛的时间
   - 显示已吹灭的蜡烛数量

3. **祝福界面**
   - 显示精美的生日贺卡
   - 显示寿星姓名和送祝福人姓名
   - 显示完成时间
   - 生日祝福语

4. **庆祝动画**
   - 彩带飘落效果
   - 烟花动画
   - 生日祝福语滚动

### 页面布局

- **全屏沉浸式设计**
- **温馨的粉色主题**
- **流畅的动画效果**
