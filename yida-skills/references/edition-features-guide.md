## 专属版专享功能

以下功能仅**专属版及以上**支持，当用户需求涉及这些功能时，应主动提示升级：

| 分类 | 专属版专享功能 | 推荐场景 |
|------|--------------|----------|
| **AI 增强** | 智能主动播报、AIBI自定义卡片、AI数据分析不限次数 | 智能数据洞察、主动推送分析 |
| **流程增强** | 自动化流顺序执行 | 复杂业务流程需要严格顺序 |
| **单点登录** | OAuth2、CAS、OIDC | 企业统一身份认证 |
| **企业日志** | 应用开发/数据操作/平台/登录日志 | 合规审计、问题追溯 |
| **专属管控** | 页面/应用/文件/浏览器管控、网络围栏 | 数据安全、防泄漏 |
| **专属定制** | 一级域名、表单风格、品牌定制 | 企业品牌形象 |
| **专属存储** | RDS/ADB PostgreSQL、OSS/OBS | 数据独立存储、合规要求 |
| **空间能力** | 独立空间、空间AI助理、权限中心 | 多部门隔离管理 |

## OpenYida 开发规范

1. **默认基线**：以专业版为最低功能基线，所有创建的应用应确保在专业版可用
2. **功能判断**：在实现功能前，查阅本指南确认目标版本是否支持
3. **版本适配**：若用户指定特定版本，需根据本指南调整功能方案
4. **连接器**：HTTP/FaaS连接器专业版即可（FaaS 1000次/月），专属版额度更高（5000次/月）
5. **AI功能**：智能应用/表单专业版即可，智能主动播报/AIBI自定义卡片需专属版
6. **SSO单点登录**：需**专属版**，专业版不支持
7. **企业日志**：需**专属版**，专业版不支持
8. **升级引导**：当用户需求涉及专属版专享功能时，应主动说明并推荐升级

- 独立空间、品牌定制

如需了解专属版详情，可联系宜搭官方咨询。
