# 宜搭用户手册核心参考文档

本文档汇总自宜搭官方用户手册，供开发时参考。

---

## 一、产品体系架构

### 1.1 页面类型

| 页面类型 | 说明 | 典型场景 |
|---------|------|---------|
| 普通表单页面 | 数据收集 | 问卷、申请、登记 |
| 流程表单页面 | 审批流程 | 请假、报销、采购 |
| 报表页面 | 数据展示 | 统计、看板、仪表盘 |
| 自定义页面 | 定制开发 | 复杂交互、数据可视化 |
| DataV 大屏 | 展示大屏 | 监控中心、指挥室 |
| 外部链接 | 外部页面 | 第三方系统集成 |

### 1.2 产品版本

| 版本 | 特点 |
|------|------|
| 免费版 | 基础功能，限人数 |
| 轻享版 | 扩展功能，中小企业 |
| 专业版 | 全部功能，企业级 |
| 专属版 | 私有化部署 |

## 二、表单设计

### 2.1 字段类型

| 字段 | 代码 | 功能 |
|------|------|------|
| 单行文本 | TextField | 短文本输入 |
| 多行文本 | TextareaField | 长文本输入 |
| 数字 | NumberField | 数值，含精度控制 |
| 单选 | RadioField | 互斥选项 |
| 多选 | CheckboxField | 多选选项 |
| 下拉单选 | SelectField | 下拉选择 |
| 下拉多选 | MultiSelectField | 多选下拉 |
| 日期 | DateField | 日期/时间 |
| 日期区间 | CascadeDateField | 开始-结束日期 |
| 人员选择 | EmployeeField | 企业成员 |
| 部门选择 | DepartmentSelectField | 组织部门 |
| 地址 | AddressField | 省市区地址 |
| 图片上传 | ImageField | 图片附件 |
| 附件上传 | AttachmentField | 通用文件 |
| 子表单 | TableField | 明细数据 |
| 流水号 | SerialNumberField | 自动编号 |
| 关联表单 | AssociationFormField | 跨表单关联 |
| 评分 | RateField | 星级评分 |
| 附件组合 | AttachmentGroupField | 批量上传 |
| 级联选择 | CascadeSelectField | 级联数据 |
| 富文本 | EditorField | 富文本编辑 |
| 国家选择 | CountrySelectField | 国家地区 |

### 2.2 字段属性

| 属性 | 说明 |
|------|------|
| 字段名称 | 显示标题 |
| 唯一标识 | 字段 ID（fieldId） |
| 字段描述 | 填写说明 |
| 占位符 | 提示文字 |
| 默认值 | 初始值 |
| 必填 | 是否必填 |
| 只读 | 是否只读 |
| 隐藏 | 是否隐藏 |

### 2.3 字段校验

| 类型 | 说明 |
|------|------|
| 必填 | 必填校验 |
| 邮箱 | 邮箱格式 |
| 手机号 | 手机号格式 |
| 身份证 | 身份证格式 |
| 自定义 | 自定义函数 |

### 2.4 公式字段

支持计算公式：

```javascript
// 常用公式示例
IF(condition, trueValue, falseValue)
CONCAT(text1, text2, ...)
LEN(text)
LEFT(text, num)
RIGHT(text, num)
TRIM(text)
NOW()
TODAY()
DATE()
YEAR(date)
MONTH(date)
DAY(date)
SUM(field1, field2, ...)
AVG(field1, field2, ...)
MAX(field1, field2, ...)
MIN(field1, field2, ...)
COUNT(field)
COUNTIF(condition)
```
